## Quick Start

### Using docker (recommended)

```bash
docker run -d --name terminal -p 3000:3000 .
```

### Using npm/yarn

0. Set up pre-commit

```bash
pre-commit install
```

1. Install dependencies:

```bash
yarn install
```

3. Run the server:

```bash
yarn dev
```
